# Facilita Currículo

Projeto desenvolvido por Sarah, Yasmin e Maria Clara, alunas do segundo ano do curso técnico em Desenvolvimento de Sistemas.

- **Sarah**: responsável pelo front-end, design e experiência do usuário.
- **Yasmin e Maria Clara**: responsáveis pelo backend do projeto.

## Descrição

Facilita Currículo é uma aplicação web que permite a criação de currículos de forma simples, moderna e acessível. O foco é proporcionar uma interface amigável e com recursos de acessibilidade, para que qualquer pessoa possa montar um currículo profissional, com campos relevantes e design responsivo.

## Funcionalidades

- Formulário para preenchimento de dados pessoais, formação, experiência profissional e resumo.
- Três modelos de currículo: simples, criativo e completo.
- Modo alto contraste e opção de aumentar a fonte para melhorar acessibilidade.
- Design responsivo para uso em diferentes dispositivos.
- Alternância fácil entre modos de acessibilidade via botões.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Flask (backend)

## Divisão de responsabilidades

- **Front-end e design:** Sarah cuidou do layout, da experiência do usuário e da implementação dos recursos de acessibilidade, como modo alto contraste e ajuste de fonte, garantindo que o site seja acessível e atraente em diferentes dispositivos.
- **Back-end:** Yasmin e Maria Clara ficaram responsáveis pela parte do servidor, processamento dos dados e geração do currículo.

## Como executar

1. Clone o repositório.
2. Antes de executar, digite no terminal os comandos: pip install flask, python app.py
3. Após os comando, no terminal aparecerá: http://127.0.0.1:5000
4. Execute no navegador
5. Abra o formulário no navegador.
6. Preencha os dados, escolha o modelo e utilize os recursos de acessibilidade conforme desejar.
